# steamfiles
Python library for parsing the most common Steam file formats.
The library has a familiar interface: load() / loads() for loading the data,
and dump() / dumps() for saving the data back to the file.

## Format support
|             | Read  | Write |
| ----------- | :---: | :---: |
| ACF         | **+** | **+** |
| appinfo.vdf | **+** | **-** |
| Manifest    | **-** | **-** |
